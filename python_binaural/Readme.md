# Python binaural
This is a python gui that uses psychoacoustic methods to add sapatial information both channels of a wav file so that is seems to be at a certain distance and angle to the listener

- Start the GUI
- Load a mono wav file via the 'Load' Button
- click into the area around the blue dot (the listener) and place the sound source (green dot) to your likings
- press 'Transform'-Button
- check out the changes in the wav sound via the 'Play'-Button
